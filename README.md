# 🚀 Library of Tampermonkey Scripts (by Calvin/River)

Добро пожаловать в коллекцию скриптов для оптимизации работы с LiveChat и Backoffice. Все скрипты поддерживают **автоматическое обновление** через этот репозиторий.

---

## 🛠 Инструкция по установке (ВАЖНО)

Чтобы у вас работали автоматические обновления, выполните следующие шаги:

1. Убедитесь, что у вас установлено расширение **Tampermonkey**.
2. В таблице ниже выберите нужный скрипт.
3. Нажмите на ссылку **"Установить"** в столбце "Ссылка".
4. В открывшемся окне Tampermonkey нажмите кнопку **"Установить"** (или "Обновить").

## 🔄 Как работают обновления?

Вам больше не нужно переустанавливать скрипты вручную. 
1. Разработчик вносит изменения в код здесь.
2. Tampermonkey сам проверяет наличие обновлений (раз в 24 часа).
3. Вы получаете новую версию автоматически!
