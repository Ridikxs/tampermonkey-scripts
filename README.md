# 🚀 Library of Tampermonkey Scripts

Добро пожаловать в библиотеку скриптов для оптимизации работы с LiveChat, Fundist и Helpdesk. Все скрипты поддерживают **автоматическое обновление** через этот репозиторий.

---

## 🎨 Фильтры и теги (Рекомендуется)
Чтобы в этом репозитории у вас появились удобные цветные теги и кнопки фильтрации по категориям (LiveChat, Fundist, Helpdesk), установите специальный скрипт оформления:
👉 **[Установить скрипт интерфейса GitHub](https://raw.githubusercontent.com/Ridikxs/tampermonkey-scripts/main/GitHubRepoTags.user.js)** *(Примечание для автора: не забудь добавить этот скрипт в репозиторий, если хочешь, чтобы другие тоже видели кнопки!)*

---

## 🛠 Инструкция по установке (ВАЖНО)
Чтобы у вас работали автоматические обновления, выполните следующие шаги:
1. Убедитесь, что у вас установлено расширение **Tampermonkey**.
2. В таблицах ниже выберите нужный скрипт.
3. Нажмите на ссылку **"Установить"**.
4. В открывшемся окне Tampermonkey нажмите кнопку **"Установить"** (или "Обновить").

## 🔄 Как работают обновления?
Вам больше не нужно переустанавливать скрипты вручную. 
1. Разработчик вносит изменения в код здесь.
2. Tampermonkey сам проверяет наличие обновлений (раз в 24 часа).
3. Вы получаете новую версию автоматически!
